# hugo-theme-moustachioed

## Homepage

Add an `_index.md` under the `content` directory. Sample config:

```toml
+++
title = "This is Moustachioed."
subtitle = "& these are some links"

[[links]]
url = "http://google.com"
name = "Google"
fa_logo = "fa-google"

[[links]]
url = "http://twitter.com"
name = "Twitter"
fa_logo = "fa-twitter"
+++
```
